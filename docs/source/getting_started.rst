Getting Started
===============

Installation
------------

Installation is strait forward. If you already have Python on your system, you can
install ``delete`` using Pip.

.. code-block::

    pip install delete-cli

For alternate installation methods see the `installation`_ page.


Basic Usage
-----------

For basic command line usage you can issue ``delete`` with no arguments. For additional information
use the ``--help`` option.

.. code-block:: bash

    delete --help


The full documentation for ``delete`` is available as a *manual page*.

.. code-block:: bash

    man delete
